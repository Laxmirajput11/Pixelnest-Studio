# PixelNest Studio

A modern and responsive landing page for **PixelNest Studio**, a fictional creative studio that helps small businesses build a professional online presence through graphic design, social media creatives, and website design.

## Live Demo

https://pixelnest-studio-plum.vercel.app/

## About

PixelNest Studio was created as part of a branding and web design assignment. The project focuses on building a complete brand identity and presenting it through a clean, modern, and user-friendly website.

## Features

* Responsive Design
* Floating Navigation Bar
* Mobile Hamburger Menu
* Interactive Services Section
* Animated Pixel Grid
* Smooth Scroll Animations
* Contact Section
* Modern UI/UX

## Services

* Graphic Design
* Social Media Creatives
* Website Design

## Design Concept

The brand is inspired by the idea that every successful business is built one step at a time—just like an image is built pixel by pixel. This concept is reflected throughout the website's visuals, animations, and overall design language.
